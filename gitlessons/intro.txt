Git - распределенная система контроля версий это система для отслеживания и ведения истории изменения файлов в вашем проекте  

Репозиторий - хранилище вашего кода и истории его изменений 

GitHub - веб сайт для размещения git-репозиториев в совместной разработке проектов 

Команды git:
1 git init - она создает новый локальный гит репозиторий у вас на пк в той директории в которой была прописана команда (команда вводится один раз в начале)
2 git add - команда для добавления файлов в систему отслеживания гита после этой команды изменения начинают отслеживаться гитом 
3 git commit -m '<comment>' - как только достигаем определенного момента в разработке то тогда мы сохраняем и комментируем изменения в локальной репозитории 
4 git remote add - это команда для того что бы связать наш локальный репозиторий с удаленным репозиторием то есть добавляет подключение к репозиторий на гитхабе
5 git push - команда для отправки всех ваших изменений на локальном репозитории в удалённый репозиторий
